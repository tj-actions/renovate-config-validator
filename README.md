## renovate-config-validator

Validate [renovate](https://github.com/renovatebot/renovate) bot configurations

```yaml
...
    steps:
      - uses: actions/checkout@v3
      - name: Renovate Config Validator
        uses: tj-actions/renovate-config-validator@v2
```
